# Saree Shop

Full-stack Saree e-commerce application with a Spring Boot backend and React frontends for customer and admin.

## Tech Stack

- **Backend:** Spring Boot, Maven, MySQL
- **Frontend (Customer):** React, React Router
- **Frontend (Admin):** React, React Router
- **Auth & Security:** JWT (JSON Web Token)
- **Build & Tools:** Git, npm, Maven

## How to Run

### Backend

cd backend
mvn spring-boot:run

Runs on: `http://localhost:3000`

### Customer Frontend

cd saree-shop-customer
npm install
npm start


Runs on: `http://localhost:3001` (or the port you configured)

## Environment & Secrets

- `.env` files and real credentials are **not committed** to the repository.
- Use local `.env` files on your machine and environment variables in production.
